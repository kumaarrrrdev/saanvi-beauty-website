# Saanvi Beauty ✨

A divine beauty salon website for women in Mysuru, Karnataka.

## 🌸 Features

- **Divine Theme**: Sanskrit blessings and traditional Indian aesthetics
- **Responsive Design**: Works on mobile, tablet, and desktop
- **WhatsApp Integration**: Direct booking via WhatsApp
- **Gallery Lightbox**: Image viewing with keyboard navigation
- **Scroll Animations**: Smooth reveal effects
- **Mobile Menu**: Hamburger navigation for small screens
